# Exercise 0.6

Create a diagram depicting the situation where the user creates a new note using the single-page version of the app.

## Solution

```mermaid
sequenceDiagram
    participant browser
    participant server

    Note right of browser: The form's submit handler calls preventDefault()
    Note right of browser: The browser creates a new note and rerenders the notes list

    browser->>server: POST https://studies.cs.helsinki.fi/exampleapp/new_note_spa
    Note right of browser: The new note is sent as JSON in the request body
    activate server
    Note left of server: The server saves the note and responds with a success status
    server-->>browser: Status code: 201 Created
    deactivate server

    Note right of browser: The page stays on the same view and does not reload
```
